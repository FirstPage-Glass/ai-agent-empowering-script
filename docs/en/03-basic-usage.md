# Part 3: Basic Usage 🎮

## Starting a Conversation 💬

1. 🖱️ Open OpenCode Desktop
2. ⌨️ You'll see an input box, just like a chat app
3. ✍️ Type your message and press Enter to send

---

## Providing Context 📎

OpenCode needs to know what you're talking about to give you a good answer.

**Providing files:** 📄
- Drag and drop files directly into the chat window
- Or click the "+" button next to the input box to select a file
- Supported formats: text files (.txt), PDF, Word (.docx), Excel (.xlsx), images, and more

**Providing images:** 🖼️
- Drag and drop images directly into the chat window
- OpenCode can "see" the content of images

**Providing web links:** 🔗
- Paste a URL directly — OpenCode will automatically read the webpage content

---

## Plan Mode vs Build Mode 📋🔨

This is one of OpenCode's most important features.

**Plan mode 📋**
- OpenCode only tells you what it *plans* to do
- It won't actually modify any files
- Best for: when you're unsure — see the plan first

**Build mode 🔨**
- OpenCode will actually execute the changes
- It will modify files, create files, organize data
- Best for: when you know exactly what you want

**How to switch:** 🔄
- There's a toggle button below the input box
- Click to switch between Plan mode ↔ Build mode
- You can also just tell it: "Give me a plan first"

> 💡 **Tip:** When you're not sure, use Plan mode to see its approach. Once you're happy, switch to Build mode and let it execute.

---

## Common Commands 🎯

| Command | What it does |
|---|---|
| `/undo` ⏪ | Undo the last change |
| `/redo` ⏩ | Redo the last undone change |
| `/clear` 🧹 | Clear the current conversation |
| `/help` ❓ | Show all available commands |

## How to Give Good Instructions 🎯

Good instructions = good results. Here are a few principles:

**1️⃣ Be specific**
- ❌ "Fix this for me"
- ✅ "Make this text more formal, suitable for a client email"

**2️⃣ Provide enough context**
- ❌ "Write an email"
- ✅ "Write an email to Mr. Zhang confirming our meeting next Monday at 10am, and include the meeting link"

**3️⃣ One thing at a time**
- ❌ "Edit this file, write an email, and organize that Excel sheet"
- ✅ Break it into three separate requests

**4️⃣ Speak up if you're not satisfied**
- ❌ (Quietly fix it yourself)
- ✅ "This version is too long, shorten it to under 200 words"

> 💡 **Think of it as a diligent intern:** the clearer your instructions, the better the results.
