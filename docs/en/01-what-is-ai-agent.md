# Part 1: What is an AI Agent? 🤖

## What is an AI Agent? 🧠

Imagine you have a **super intern** 👨‍💻. They:
- 👂 Understand what you say (natural language)
- 🛠️ Can actually do things for you (not just chat)
- 👀 Show you what they're doing (you can approve or reject)

**AI Agent = AI Brain 🧠 + Ability to Take Action 🛠️**

> 💡 **OpenCode is open source** (MIT license, 172k+ GitHub stars) — the most popular open-source AI coding agent. It's free, transparent, and you can inspect the code yourself.

---

## How is it Different from ChatGPT? 🤔

| | ChatGPT 💬 | OpenCode (AI Agent) 🦾 |
|---|---|---|
| 🎯 What it does | Answers questions, chats | Handles files, organizes data, edits documents |
| 📦 What you get | Text answers | Actual results (edited files, organized data) |
| 🔄 Interaction | One question, one answer | You watch it work, can pause or revise at any time |
| ✉️ Example | "Write an email" → gives you text | "Write an email" → opens the file, writes it, shows you, saves when you approve |

> 💡 **One-liner: ChatGPT tells you what to do. OpenCode does it for you.**

## Three Ways to Use OpenCode 🖥️

| Version | Who it's for | How to use |
|---|---|---|
| **Desktop App** 🖼️ | **Everyone** 👤 | Opens like any app, has a graphical interface — easiest option |

> 🖼️ **The Desktop App is designed for you** — just double-click the icon like Chrome or Word. Best choice for first-time users.

---

## Key Concepts 📌

**1️⃣ Conversation 💬**
Every interaction with OpenCode is a conversation. Just like WhatsApp — one message after another, and it remembers what you said before.

**2️⃣ Context 📎**
You can give OpenCode files, images, and web pages so it understands what you're talking about.
> ✋ "Help me edit this document" → you need to give it the document first

**3️⃣ Plan mode vs Build mode 📋🔨**
- **Plan mode** 📋: OpenCode tells you what it *plans* to do, but doesn't actually do it
- **Build mode** 🔨: OpenCode actually executes the changes

> 💡 **Tip:** When unsure, use Plan mode first. Review the plan, then let it execute.

**4️⃣ /undo ⏪**
If OpenCode makes a change you don't like, type `/undo` to revert. No need to worry about breaking things.
