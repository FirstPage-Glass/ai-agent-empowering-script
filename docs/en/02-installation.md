# Part 2: Installation & Setup 🚀

## Step 1: Run the Installation Script 📥

We've prepared a one-click installation script that will install OpenCode Desktop along with other useful tools.

### Mac Users 🍎

Open Terminal and paste the following command, then press Enter:

```bash
curl -fsSL https://raw.githubusercontent.com/FirstPage-Glass/ai-agent-empowering-script/main/install-dev-tools-macos.sh | bash
```

> ⚠️ If prompted for a password, enter your computer password (characters won't show as you type — this is normal).

### Windows Users 🪟

1. Search for "PowerShell" in the taskbar
2. Right-click "Windows PowerShell" and select "Run as Administrator"
3. Paste the following command and press Enter:

```powershell
irm https://raw.githubusercontent.com/FirstPage-Glass/ai-agent-empowering-script/main/install-dev-tools-windows.ps1 | iex
```

---

## Step 2: Start Using It 🎉

After installation, double-click the OpenCode Desktop icon.

**That's it.** No registration, no API key, no credit card required.

You'll see a chat window with free models ready to use. Just type something:

```
You: Hello 👋
OpenCode: Hello! How can I help you?
```

> 🎊 **Congratulations, you're ready to go!**

## Advanced: Using More Powerful Models (Optional) 🚀

If you'd like to try more powerful models later (like Claude, GPT-5, etc.), you have two options:

### 1️⃣ OpenCode Zen 🟢
OpenCode's official paid model service, quality guaranteed.
In OpenCode Desktop, type `/connect` and press Enter — it will guide you through setup.

### 2️⃣ Your Own API Key 🔑
If you already have an OpenAI or Anthropic API key,
you can paste it in the Settings page.

> 💡 **What's an API Key?** Think of it as a password that lets OpenCode connect to AI services. But you don't need one right now — the free built-in models are enough to get started.
