# Part 4：日常應用場景

## 場景一：寫 Email

**你對它說：**
```
幫我寫一封 email 給客戶王小姐，內容包括：
1. 感謝她上週參加我們的產品說明會
2. 附上產品目錄 PDF
3. 詢問她是否有興趣安排一次一對一示範
4. 語氣要專業但友善
```

**opencode 會做：**
- 寫好完整的 email 內容給你看
- 你可以要求修改語氣、增減內容
- 滿意後直接複製貼到 Outlook/Gmail

---

## 場景二：修改文章

**你對它說：**
```
這段文字太囉嗦了，幫我濃縮成 100 字以內：

[貼上你的文字]
```

**或者：**
```
幫我把這段文字改成更正式的商業語氣：

[貼上你的文字]
```

---

## 場景三：翻譯

**你對它說：**
```
把以下內容翻譯成繁體中文：

[貼上英文內容]
```

**或者：**
```
幫我把這份文件翻譯成英文，保留原本的格式：

[貼上文件或拖曳檔案]
```

---

## 場景四：整理資料

**你對它說：**
```
我有一個 Excel 檔案，裡面是本月客戶訂單資料。
幫我：
1. 找出消費金額最高的前 10 名客戶
2. 計算每位客戶的平均訂單金額
3. 整理成一個表格

[拖曳 Excel 檔案到對話視窗]
```

---

## 場景五：摘要重點

**你對它說：**
```
幫我 summarize 這篇文章，列出 5 個重點：

[貼上文章內容或網址]
```

---

## 場景六：腦力激盪

**你對它說：**
```
我們下個月要辦一場線上研討會，主題是「AI 如何幫助中小企業」。
幫我想 10 個吸引人的活動名稱，以及 3 種不同的活動形式。
```

---

## 場景七：研究調查

**你對它說：**
```
幫我研究 competitor X 這家公司：
1. 他們的主要產品是什麼？
2. 定價大約多少？
3. 他們的目標客戶是誰？
4. 跟我們相比，優缺點是什麼？
```

> 💡 opencode 可以上網搜尋資料，但建議你還是要自己驗證資訊的正確性。
