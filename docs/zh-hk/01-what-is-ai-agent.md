# Part 1：什麼是 AI Agent？ 🤖

## 什麼是 AI Agent？ 🧠

想像你有一位**超級實習生** 👨‍💻。他：
- 👂 聽得懂你說的話（自然語言）
- 🛠️ 可以動手幫你做事（不只是聊天）
- 👀 會讓你看他做了什麼（你可以批准或拒絕）

**AI Agent = AI 大腦 🧠 + 動手做事的能力 🛠️**

> 💡 **OpenCode 是開源軟件**（MIT 授權，GitHub 172k+ 星）— 最受歡迎的開源 AI Agent。完全免費、透明，程式碼可以自己檢查。

---

## 跟 ChatGPT 有什麼不同？ 🤔

| | ChatGPT 💬 | OpenCode (AI Agent) 🦾 |
|---|---|---|
| 🎯 做什麼 | 回答問題、聊天 | 動手幫你處理文件、整理資料、修改檔案 |
| 📦 給你什麼 | 文字答案 | 實際成果（修改好的檔案、整理好的資料） |
| 🔄 互動方式 | 一問一答 | 你看著它做，可以中途叫它停或修改 |
| ✉️ 例子 | 「幫我寫一封 email」→ 給你一段文字 | 「幫我寫一封 email」→ 幫你開好檔案寫完，給你看過，你同意就儲存 |

> 💡 **一句話總結：ChatGPT 是「告訴你怎麼做」，OpenCode 是「幫你做好」**

## OpenCode 的三種使用方式 🖥️

| 版本 | 適合誰 | 使用方式 |
|---|---|---|
| **Desktop App** 🖼️ | **每個人** 👤 | 像一般應用程式一樣開啟，有圖形介面，最簡單 |
| **VS Code Extension** 🧩 | **每個人** 👤 | 裝在 VS Code（一個免費的文字編輯器）裡面使用 |
| Terminal (TUI) ⌨️ | 開發者 / 進階使用者 | 命令列模式 |

> 💡 **VS Code 是什麼？** 它只是一個**免費的文字編輯器**，不是只有工程師才能用。你平時改文件、寫筆記、看資料都可以用它，就像用 Word 一樣。

> 🖼️ **Desktop App 是為你設計的** — 像開啟 Chrome、Word 一樣，雙擊圖示就能開啟，最適合第一次使用的人。

## 重要概念 📌

**1️⃣ Conversation（對話）💬**
你與 OpenCode 的每一次互動稱為一個 conversation。就像 WhatsApp 一樣，一句接一句，它會記得之前說過的話。

**2️⃣ Context（背景資料）📎**
你可以提供檔案、圖片、網頁給 OpenCode，讓它了解你在說什麼。
> ✋ 「幫我修改這份文件」→ 你需要先把文件給它看

**3️⃣ Plan mode vs Build mode 📋🔨**
- **Plan mode** 📋：OpenCode 只會告訴你它打算怎麼做，不會實際動手
- **Build mode** 🔨：OpenCode 會實際執行

> 💡 **建議：** 不確定的時候先用 Plan mode，看過計劃再讓它執行。

**4️⃣ /undo（還原）⏪**
如果 OpenCode 做了你不想要的修改，輸入 `/undo` 就會還原。不用擔心弄壞東西。
