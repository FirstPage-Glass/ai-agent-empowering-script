# Part 6：常見問題 ❓

### Q：我完全不會寫程式，可以用嗎？ 🤔
**A：可以。** opencode Desktop 有圖形介面，像一般應用程式一樣操作。你只需要會打字就可以了。

---

### Q：要付費嗎？ 💰
**A：不用。** opencode Desktop 裝完就有免費模型可以直接使用。

如果你之後想要更強大的模型，有這些選擇：

| 方案 | 費用 | 說明 |
|---|---|---|
| 🆓 **免費模型** | **免費** ✅ | 裝完就有，直接用到 |
| 🟢 **opencode Go** | 首月 $5，之後 $10/月 | 低成本訂閱，包含多個強大開源模型，額度充裕 |
| 🔵 **opencode Zen** | 按量計費 | 用多少付多少，包含 Claude、GPT-5 等頂級模型 |
| 🔑 **自備 API Key** | 看你用哪家 | 用自己的 OpenAI / Anthropic 等 API Key |

> 💡 **建議：** 先免費試用，覺得有需要再考慮升級。opencode Go 首月只要 $5 美元，很適合想試試更強模型的人：[opencode.ai/zht/go](https://opencode.ai/zht/go)

---

### Q：我的資料安全嗎？ 🔒
**A：這取決於你使用的模型。** opencode 本身不會儲存你的資料。免費模型可能會收集資料用於改進，建議不要上傳公司機密資料，並遵守公司的資訊安全政策。

---

### Q：AI 會出錯嗎？ ⚠️
**A：會。** AI 不是完美的，它可能會：
- 🤷 理解錯誤
- ❌ 給出錯誤的資訊
- 🤪 產生不合理的內容

**所以重要的事情一定要自己再確認一遍。**

---

### Q：opencode 跟 ChatGPT 有什麼不同？ 🤷
**A：ChatGPT 是「告訴你怎麼做」，opencode 是「幫你做好」。** opencode 可以直接修改檔案、整理資料、執行動作，而不只是給你建議。
